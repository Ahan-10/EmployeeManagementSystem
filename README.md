# Employee Management System

A web-based Employee Management System developed using Java Servlets, JSP, MySQL, JDBC, and Apache Tomcat.

## Features

- Admin Login
- Employee Login
- Add Employee
- Update Employee
- Delete Employee
- Employee Profile Management
- Payslip Generation
- Session Management
- Email Notifications
- Role-Based Access Control

## Technologies Used

- Java
- JSP
- Servlets
- JDBC
- MySQL
- Apache Tomcat 9
- HTML
- CSS
- JavaScript

## Project Structure

src/main/java
├── dao
├── model
├── servlet
└── util

src/main/webapp
├── login.jsp
├── admin-dashboard.jsp
├── add-employee.jsp
├── edit-employee.jsp
├── employee-profile.jsp
└── payslip.jsp

## How to Run

1. Install Java JDK 21+
2. Install Apache Tomcat 9
3. Install MySQL
4. Create the database and tables
5. Import the project into Eclipse
6. Deploy on Tomcat
7. Open:

http://localhost:8080/EmployeeManagementSystem/login
